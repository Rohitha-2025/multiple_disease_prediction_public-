# 🩺 Multiple Disease Prediction System using Machine Learning

This project is a **Streamlit-based web application** that predicts the likelihood of multiple diseases using Machine Learning models.

---

## 🚀 Live Demo

🔗 **Deployed App:**
https://d7tnggxkcvyyzrrsypnbjn.streamlit.app

---

## 📌 Features

* 🧠 **Diabetes Prediction**
* ❤️ **Heart Disease Prediction**
* 🧬 **Parkinson’s Disease Prediction**
* 🌐 User-friendly web interface using Streamlit
* ⚡ Instant predictions based on input data

---

## 🛠️ Technologies Used

* Python 🐍
* Streamlit
* Scikit-learn
* NumPy
* Pandas

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/multiple_disease_prediction_public.git
```

2. Navigate to the project folder:

```
cd multiple_disease_prediction_public
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```

---

## 📊 How It Works

* The user inputs medical parameters
* The trained ML model processes the data
* The system predicts whether the disease is likely or not
